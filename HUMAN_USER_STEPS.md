# Human User Steps - V2

This workflow is designed to be as simple as possible for Ricky/cousin.

## Do this for each item

1. Assign SKU.
2. Put SKU on item, bag, or bin.
3. Use eBay AI/photos to create draft.
4. Put the same SKU in eBay Custom Label/SKU.
5. Save draft.
6. Do not publish.
7. Paste SKU + Draft URL into `DRAFT_REVIEW_QUEUE`.
8. Set `Status` to `READY_FOR_AGENT_REVIEW`.

## Stop there

The Agent handles:

- item review
- draft audit
- sold-comp research
- low/average/max price research
- recommended list price
- warnings and blockers
- spreadsheet output

## Human publishes only after review

Publish only when the item is marked ready and the human agrees with price/details.
